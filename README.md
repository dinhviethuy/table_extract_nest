# Table Extractor API (NestJS)

## Cài đặt & Chạy nhanh

1. Cài đặt dependencies:
   ```bash
   pnpm install
   ```

2. Cấu hình môi trường (Tạo file `.env` từ `.env.example` và điền đủ thông tin):
   ```bash
   cp .env.example .env
   ```

3. Chạy ứng dụng:
   - **Development**: `pnpm run start:dev`
   - **Production**: `pnpm run build && pnpm run start:prod`

4. Chạy kiểm thử (Unit Tests):
   ```bash
   pnpm run test
   ```
